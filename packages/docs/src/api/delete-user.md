# Delete a user

## Request

```js
{
    type: POST,
    url: "http://localhost:3030/users",
    Authorization: "Bearer <TOKEN-HERE>",
}
```

## Response

-   Return type : JSON

```JSON
{
  "_id": "5fc8ee87b590f20f2483e175",
  "email": "j@k.com",
  "createdAt": "2020-12-03T13:56:23.272Z",
  "updatedAt": "2020-12-03T13:56:23.272Z",
  "__v": 0
}
```
